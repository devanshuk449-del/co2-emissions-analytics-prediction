# CO₂ Emissions Analytics & Prediction

## Overview

This project analyzes global CO₂ emissions data from the IEA/EDGAR dataset and predicts future emission levels using Machine Learning.

The application provides an interactive Streamlit dashboard for:

* Annual emissions trend analysis
* Monthly seasonality analysis
* Sector-wise emissions analysis
* Random Forest regression modeling
* Future emissions prediction

## Features

* Interactive dashboard
* CO₂ emissions trend visualization
* Top emitting sectors analysis
* Feature importance analysis
* Machine Learning forecasting
* Streamlit web application

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Model

Random Forest Regressor

Evaluation Metrics:

* R² Score
* RMSE

## Dataset

IEA/EDGAR Global CO₂ Emissions Dataset (1970–2023)
